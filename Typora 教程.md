## Typora 教程

### 标题级别

\# 一级标题 快捷键为 Ctrl + 1
\## 二级标题 快捷键为 Ctrl + 2
......
\###### 六级标题 快捷键为 Ctrl + 6

### 引用文字

\> + 空格 + 引用文字

### 无序列表

使用 * + - 都可以创建一个无序列表

- AAA
- BBB
- CCC

### 有序列表

使用 1. 2. 3. 创建有序列表

1. AAA
2. BBB
3. CCC

### 任务列表

\ -[ ] 不勾选
\ -[x] 勾选

### 数学表达式

按下 `$$`，然后按下回车键，即可进行数学公式的编辑。

### 插入表格

输入 `| 表头1 | 表头2 |`并回车。即可创建一个包含2列表。快捷键 `Ctrl + T`弹出对话框。

### 目录（TOC）

输入 `[ toc ]` 然后回车，即可创建一个“目录”。TOC从文档中提取所有标题，其内容将自动更新。
